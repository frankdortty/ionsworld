Web Design Masterclass

The web design training course is a 5 weeks course which aims to give you the knowledge and skills to start developing your own websites with the latest technologies. So this course focuses on the practical application of the technologies used in web development.

Learn the coding languages and technologies used in web design so that you can have the practical skills and knowledge to start developing your own websites. This course will also cover the emerging trends of the internet, how these come to be, and how they are constantly refined. You will learn how to keep on top of current trends to ensure that your websites are always designed according to modern standards
Course Highlights
  • Learn the skills you need to begin your journey into becoming a professional web developer and open yourself up to countless career opportunities.
  • Gain the practical skills to start designing webpages
  • Technologies to be learnt includes HTML5, CSS3, Bootstrap & WordPress
  • Become job ready with internship opportunities available for students
Outline

In this course you will learn:
  • Introduction to web design & development
  • HTML5 & CSS3
  • Planning a Website
  • Modern Responsive Web Design
  • Build mobile responsive website with Bootstrap
  • Setting up a web server
  • Content management systems
  • Designing websites using WordPress
  • Programming fundamentals
  • Going Live
  • Web Trends
Who Should Attend

This course is ideal for beginners and intermediate students who want to gain practical skills to start developing web content. No prior knowledge or experience is required.
